# MicrobitPumpkin
Code for a Micro:Bit Jack O' Lantern complete with movement.

You need 11 Micro:Bits to complete this project.
<UL>
  <LI>2 Eyes</LI>
  <LI>2 Top Teeth</LI>
  <LI>2 Bottom Teeth</LI>
  <LI>Nose</LI>
  <LI>Left Mouth</LI>
  <LI>Left Mouth 2</LI>
  <LI>Right Mouth 2</LI>
  <LI>Right Mouth</LI>
  
  The nose sends the radio signal to synchronize the moving pieces. Allign the Micro:Bits upside down to create the face.
